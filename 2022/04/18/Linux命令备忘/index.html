others

  • 查看当前单词的 man 手册页：Shift+k, 如果想指定跳转到库函数的man,需要用3k
  • 查看命令的帮助： :h {key}
  • 从widows复制到wsl2: ctrl+shift+v

移动

  • 查看所有移动的命令: h motion.txt

  • 左：h 下：j 上：k 右：l

  • 向下翻页： ctrl+d 向上翻页：ctrl+b

  • 单词移动:

    • 有两种word: word和WORD

      word表示：

      • 字母, 数字和下划线的序列(正则表达式：\w+)或
      • 其他非空白字符的序列(正则表达式：[^\s\w]+)

      WORD表示：

      • 一系列非空白字符(正则表达式：\S+

      比如:help usr_03.txt

      • 有5个word: :, help,usr_03, .,txt
      • 有2个WORD: :help, usr_03.txt

    • vim w,e,b 表示word的移动, 而shift+key表示WORD的移动

    • w表示前进到“word”的开头,shift+w表示前进到“WORD”的开头

    • b表示后退到word的开头,shift+b表示后退到WORD的开头

    • e表示前进到word的结尾, shift+e表示前进到WORD的结尾

  • 向后移动一个段落:shift+[ 向前移动一个段落: shift+]

  • 当光标移动到括号时()或[]或{}, 在括号之间跳转:%

  • 向下查找: / 向上查找: ?

  • 跳到行首: 0, 跳到行尾:$

  • 把光标当前所在行放到页中间, 最上方, 最底下:zz, zt, zb

dd命令

dd 可从标准输入或文件中读取数据，根据指定的格式来转换数据，再输出到文件、设备或标准输出。

注意：指定数字的地方若以下列字符结尾，则乘以相应的数字：b=512；c=1；k=1024；w=2

参数注释：

  1. if=文件名：输入文件名，缺省为标准输入。即指定源文件。< if=input file >

  2. of=文件名：输出文件名，缺省为标准输出。即指定目的文件。< of=output file >

  3. ibs=bytes：一次读入bytes个字节，即指定一个块大小为bytes个字节。

    obs=bytes：一次输出bytes个字节，即指定一个块大小为bytes个字节。

    bs=bytes：同时设置读入/输出的块大小为bytes个字节。

  4. count=blocks：仅拷贝blocks个块，块大小等于ibs指定的字节数。

    /dev/zero

像/dev/null一样， /dev/zero也是一个伪文件.

/dev/zero主要的用处是用来创建一个指定长度用于初始化的空文件, 它会无穷尽地提供0

shell中的命令替换

命令替换与变量替换差不多，都是用来重组命令行的，先完成引号里的命令行，然后将其结果替换出来，再重组成新的命令行。

shell中命令替换符有两种``与$()

当您通过 $( CMD ) 这样的方式来执行CMD 这个命令时，它的输出结果会替换掉 $( CMD ) 。例如，如果执行 for file in $(ls) ，shell首先将调用ls ，然后遍历得到的这些返回值。
``与$()两者都起到了同样的效果，都是起到命令替换的作用。但是，两者还是有细微的不同：

  • 反单引号适用于任何类unix平台，他的适用性比较高。但是$符号却不是。
  • $()更直观。

screen：多重视窗管理程序

  • 进入screen： screen 回到之前的screen： screen -r

  • 创建新的窗口： ctrl+a c 即Ctrl键+a键，之后再按下c键

  • 前后切换窗口： ctrl+a n 切换到下一个窗口， ctrl+a p 切换到前一个窗口

  • 关闭当前窗口： ctrl+a k

  • 分屏操作：

    • 上下分屏：ctrl + a 再按shift + s
    • 左右分屏： ctrl+a 再按shift+|
    • 切换可视区域：ctrl+a tab
    • 关闭当前可视区域（并没有关闭窗口）： ctrl+a shift+k

  • 列出所有窗口： ctrl+a shift+"

  • 为当前窗口设置标题： ctrl+a shift+a
